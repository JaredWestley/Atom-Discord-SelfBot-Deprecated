## 📖 Overview

**AtomSB** was my first proper attempt at a large coding project back in 2020-22, it's designed to give a user account on discord access to bot commands and features.

The application consists of:

- 📱 Over 100+ commands.
- 🧠 Python-based codebase.

The version I have uploaded here is V1.4, at one point I got up to V2 but has been lost to time. The Final version V2 had over 225+ commands. For production, I used a tool called [PyArmor](https://pyarmor.dashingsoft.com/) to obfuscate my code. This allowed me to sell the selfbot without worrying about reverse engineering and reduced the compelxity of distribution.

Most of the links, features and functions are fundamentally broken due to the age and lack of support for old versions of software. I have added some handlers to allow you to get it running up to the dashboard screen. And I have also tried to modularise it for easier reviewing.

This project was also combined with a launcher that would provide easier viewing of commands, changing config and downloading the latest version of the selfbot. I have lost the code to this C# project unfortunately:
